Xplorer

Internet Explorer energy, X product soul — the browser built around Grok. Search, chat, build apps, and browse the web with an AI companion that lives inside every page.

Grok-native search

Start every session from a Grok-powered search home. Ask questions, get web results, images, and videos — then open answers on grok.com with full context carried over.

AI on every site

A Grok toolbar on grok.com, x.com, and Grokipedia keeps Chat, Build, Web, and Wiki one click away. The Grok It button on any page sends what you are reading straight into Grok Web.

Build mini-apps

Describe an app in plain language and Xplorer hosts it on localhost with its own runtime. Iterate in chat, open in a tab, and keep building without leaving the browser.

Agent-ready by default

Remote debugging and an agent gateway run out of the box so coding agents can drive tabs, capture screenshots, and automate workflows — no flags or extensions required.

Why not just use Chrome?

  • AI-first home — new tabs open Grok search, not a blank page or ad feed.
  • Native Grok UI — chat and apps are served inside the browser, not a slow extension popup.
  • X + Grok together — jump between x.com, X Chat, grok.com, and Grokipedia from one toolbar.
  • Built for agents — CDP on 9333 and a local gateway on 9334; discovery at ~/.xplorer/gateway.json.
