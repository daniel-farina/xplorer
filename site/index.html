For developers & agents

An API the agent layer deserves.

Xplorer exposes fast, high-level primitives over a local gateway — plus raw CDP for everything you already built.

connect.py 
from xplorer_sdk import Browser

b = Browser()                  # local gateway, token-authed
page = b.new_tab("https://example.com")

print(page.text())             # clean extracted text, one round trip
page.click("a#more")
tree = page.axtree()           # accessibility tree for grounding
